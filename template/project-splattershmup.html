Project overview

Splattershmup is a shoot-em-up game with the goal of creating splatter paint art through gameplay styled after famous artist Jackson Pollock. Moving the player around and destroying enemy ships create paint trails. After each game the player can then view their generated artwork and share it through various social media sites. We made this project in the IGM Production Studio class at RIT with the ambitious goal of releasing the game at the end of the term. Although we did not make it to that release goal we are continuing to work on it and are expecting to complete it Spring 2015.
