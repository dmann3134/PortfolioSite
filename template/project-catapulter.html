Project overview

Catapulter is a small mobile side project created for the exploration of turn based networking for mobile devices. It is a very simple catapult game with the objective of eliminating the VIPs in the other player's castle. I am exploring the use of the GooglePlay API to more easily facilitate credential usage and data transfer.
